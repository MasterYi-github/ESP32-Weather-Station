# ESP32 温湿度监测站

基于 ESP32 的物联网温湿度监测系统，支持本地 OLED 显示和手机/电脑网页远程查看。

## 功能特点
- 🌡️ 实时采集温湿度（AHT20 传感器）
- 📟 1.3寸 OLED 本地显示（SH1106 驱动）
- 🌐 网页服务器远程查看数据
- 🔘 手机浏览器远程控制 LED
- 🔄 数据每 2 秒自动刷新

## 硬件清单
- ESP32 开发板
- AHT20 温湿度传感器（I2C）
- 1.3寸 OLED 屏幕（SH1106）
- 杜邦线若干

## 软件依赖
- Arduino IDE
- 库：U8g2, Adafruit_AHTX0, WiFi, WebServer

## 使用方法
1. 修改代码中的 WiFi 名称和密码
2. 上传到 ESP32
3. 打开串口监视器，获取 IP 地址
4. 手机/电脑连接同一 WiFi，浏览器访问该 IP

## 遇到的问题及解决
| 问题 | 原因 | 解决方法 |
|------|------|----------|
| OLED 显示乱码 | 1.3寸屏用 SH1106 而非 SSD1306 | 改用 U8g2 库的 SH1106 驱动 |
| WiFi 连不上 | 手机热点默认 5GHz | 切换到 2.4GHz 频段 |
